# AI 공부 도우미 앱 - 디자인 문서

## 앱 개요
AI 기반 공부 앱으로, 사용자가 시험문제를 촬영하면 AI가 문제를 분석하고 풀이 과정을 제공합니다. 또한 사용자가 풀었던 문제들을 분석하여 어떤 유형의 문제가 많이 나오는지 통계를 제공합니다.

## 핵심 기능
1. **문제 촬영 및 인식**: 카메라로 시험문제를 촬영하거나 갤러리에서 이미지 선택
2. **AI 문제 풀이**: AI가 문제를 분석하고 단계별 풀이 과정 제공
3. **문제 히스토리**: 풀었던 문제들을 저장하고 다시 확인
4. **문제 유형 분석**: 풀었던 문제들의 유형을 분석하여 통계 제공

## 화면 구조 (모바일 세로 모드 9:16)

### 1. 홈 화면 (Home)
**주요 콘텐츠:**
- 상단: 앱 로고 및 환영 메시지
- 중앙: 큰 "문제 촬영하기" 버튼 (카메라 아이콘)
- 하단: 최근 풀이한 문제 3개 미리보기 (썸네일 + 날짜)

**기능:**
- 문제 촬영하기 버튼 → 카메라/갤러리 선택 시트 표시
- 최근 문제 카드 탭 → 해당 문제 상세 화면으로 이동

**레이아웃:**
- 한 손으로 조작 가능하도록 주요 버튼은 화면 중앙~하단에 배치
- 카드 형태의 최근 문제 리스트는 가로 스크롤 가능

### 2. 문제 히스토리 화면 (History)
**주요 콘텐츠:**
- 상단: "문제 히스토리" 타이틀
- 리스트: 풀었던 문제들의 목록 (썸네일, 날짜, 문제 유형)
- 각 항목: 문제 이미지 썸네일 + 풀이 날짜 + 문제 유형 태그

**기능:**
- 리스트 항목 탭 → 문제 상세 화면으로 이동
- 풀 투 리프레시로 목록 새로고침
- 무한 스크롤로 이전 문제들 로드

**레이아웃:**
- 세로 스크롤 리스트 형태
- 각 카드는 좌측에 썸네일, 우측에 정보 표시

### 3. 통계 화면 (Statistics)
**주요 콘텐츠:**
- 상단: "문제 유형 분석" 타이틀
- 통계 카드들:
  - 총 풀이한 문제 수
  - 가장 많이 푼 문제 유형 Top 5 (막대 그래프)
  - 최근 7일간 풀이 추세 (라인 차트)
  - 과목별 분포 (파이 차트)

**기능:**
- 각 통계 카드는 탭하여 상세 정보 확인 가능
- 기간 필터 (최근 7일, 30일, 전체)

**레이아웃:**
- 세로 스크롤 가능한 카드 레이아웃
- 각 통계는 독립적인 카드로 구성

### 4. 문제 상세 화면 (Problem Detail)
**주요 콘텐츠:**
- 상단: 뒤로가기 버튼
- 문제 이미지 (확대 가능)
- AI 풀이 섹션:
  - 문제 유형 태그
  - 단계별 풀이 과정 (마크다운 형식)
  - 핵심 개념 설명
- 하단: 삭제 버튼

**기능:**
- 이미지 핀치 줌으로 확대/축소
- 풀이 내용 복사 가능
- 문제 삭제 기능

**레이아웃:**
- 세로 스크롤 가능
- 이미지는 화면 너비에 맞춤
- 풀이 내용은 읽기 편한 여백과 줄간격

### 5. 문제 촬영/업로드 화면 (Camera/Upload)
**주요 콘텐츠:**
- 카메라 프리뷰 또는 이미지 선택 인터페이스
- 촬영/선택 버튼
- 미리보기 및 확인 화면

**기능:**
- 카메라로 직접 촬영
- 갤러리에서 이미지 선택
- 촬영/선택 후 이미지 확인 및 재촬영 옵션
- 확인 후 AI 분석 시작

**레이아웃:**
- 전체 화면 카메라 프리뷰
- 하단에 촬영 버튼 (큰 원형 버튼)
- 갤러리 버튼은 좌측 하단

## 주요 사용자 플로우

### 플로우 1: 문제 촬영 및 풀이 받기
1. 홈 화면에서 "문제 촬영하기" 버튼 탭
2. 바텀 시트에서 "카메라로 촬영" 또는 "갤러리에서 선택" 선택
3. 문제 이미지 촬영/선택
4. 이미지 확인 화면에서 "분석 시작" 버튼 탭
5. 로딩 화면 (AI 분석 중...)
6. 문제 상세 화면으로 이동 (풀이 결과 표시)

### 플로우 2: 이전 문제 다시 보기
1. 하단 탭바에서 "히스토리" 탭 선택
2. 문제 목록에서 원하는 문제 카드 탭
3. 문제 상세 화면에서 풀이 내용 확인

### 플로우 3: 문제 유형 통계 확인
1. 하단 탭바에서 "통계" 탭 선택
2. 각종 통계 카드 확인
3. 특정 통계 카드 탭하여 상세 정보 확인

## 색상 선택 (브랜드 컬러)
- **Primary (메인 컬러)**: `#6366F1` (인디고 블루) - 학습과 집중을 상징
- **Background (배경)**: 
  - Light: `#FFFFFF` (화이트)
  - Dark: `#0F172A` (다크 블루)
- **Surface (카드 배경)**:
  - Light: `#F8FAFC` (라이트 그레이)
  - Dark: `#1E293B` (다크 그레이)
- **Foreground (텍스트)**:
  - Light: `#0F172A` (다크 블루)
  - Dark: `#F1F5F9` (라이트 그레이)
- **Muted (보조 텍스트)**:
  - Light: `#64748B` (그레이)
  - Dark: `#94A3B8` (라이트 그레이)
- **Success (성공)**: `#10B981` (그린) - 문제 해결 완료
- **Warning (경고)**: `#F59E0B` (오렌지) - 주의 필요
- **Error (오류)**: `#EF4444` (레드) - 오류 발생

## 디자인 원칙
1. **간결함**: 학습에 집중할 수 있도록 불필요한 요소 최소화
2. **직관성**: 첫 사용자도 쉽게 이해할 수 있는 명확한 UI
3. **접근성**: 충분한 대비와 큰 터치 영역으로 사용성 향상
4. **일관성**: iOS HIG를 따르는 네이티브 앱 느낌
5. **피드백**: 모든 액션에 즉각적인 시각적 피드백 제공

## 기술적 고려사항
- **로컬 우선**: 문제 히스토리는 AsyncStorage에 저장 (빠른 접근)
- **AI 처리**: 서버의 내장 LLM 사용 (사용자 API 키 불필요)
- **이미지 처리**: 서버의 S3 스토리지에 업로드 후 AI 분석
- **오프라인 지원**: 이전에 풀었던 문제는 오프라인에서도 확인 가능
- **사용자 인증**: 선택 사항 (로컬 앱으로 먼저 구현, 추후 클라우드 동기화 시 필요)
